# react-native-ios-meshgradient

An implementation of MeshGradient in React Native

## Installation

```sh
npm install react-native-ios-meshgradient
```

## Usage

```js
import { IosMeshgradientView } from "react-native-ios-meshgradient";

// ...

<IosMeshgradientView color="tomato" />
```
